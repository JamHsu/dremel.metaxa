Metaxa version

Single-JVM, single jar, 100% pure implementation of Google BigQuery/Dremel system.

==============

src => java sources
lib => binary java liraries
data => Static data and other non-java files
test* => generally files needed only for testing
test => java unit tests
testdata => data files and other non-java files for unit tests
testlib => binary java libraries needed only for unit tests
tools => tools used to build and generally work on the project.
requirements.txt => requirements
design.txt => design
coding_guidelines => Coding styling and general coding guidelines
melnik_VLDB10 => the bible for the project. The academic papers describing bigquery/dremel which metaxa implementing
